Phelem.js

Awesome web experience powered by physics

Phelem.js (short for physical elements) is a simple JavaScript library that transforms DOM elements into (simulated) physical objects which are then affected by events such as scrolling, dragging, writing and mouse movements; the effect looks and feels awesome!

Phelem.js is currently in version 0.0.1 and in development!

Take a look at the Phelem's GitHub repository for more information, the latest version and the download link.

Click here for a demo where the document's body is Phelemized.
